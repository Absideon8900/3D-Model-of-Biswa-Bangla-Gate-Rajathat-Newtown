# 3D Model of Biswa Bangla Gate, Rajarhat, Newtown

A 3D Model built on MS 3D Builder

## 📋 Description

This project features a detailed 3D model of the iconic Biswa Bangla Gate located in Rajarhat, Newtown, Kolkata. The model has been created using Microsoft 3D Builder, capturing the architectural essence of this landmark.

## 🏗️ About Biswa Bangla Gate

Biswa Bangla Gate is a prominent monument and cultural landmark in Kolkata, representing the rich heritage and architectural beauty of West Bengal. This 3D model aims to preserve and showcase its intricate design digitally.

## 🛠️ Tools & Technologies

- **3D Modeling Software**: Microsoft 3D Builder
- **Format**: 3D model files

## 🚀 Getting Started

### Prerequisites
- Microsoft 3D Builder or a compatible 3D viewing application
- Sufficient storage space for 3D model files

### Installation & Usage

1. Clone this repository:
   ```bash
   git clone https://github.com/Absideon8900/3D-Model-of-Biswa-Bangla-Gate-Rajathat-Newtown.git
   ```

2. Navigate to the project directory:
   ```bash
   cd 3D-Model-of-Biswa-Bangla-Gate-Rajathat-Newtown
   ```

3. Open the 3D model files using Microsoft 3D Builder or compatible 3D viewers:
   - Windows 3D Viewer
   - Autodesk Viewer
   - Other 3D model viewers
